# Marmoset Pro Userscripts

## Installation
1. Install Tampermonkey browser extension
2. Create new script for each userscript file
3. Copy and paste the entire code content
4. Save the script (Ctrl+S)
5. Update the extension permissions Tampermonkey tells you to update

## Scripts Included
### Marmoset Plus Improved
- Adds public and release test case columns to course page
- Auto-refreshes project pages during testing
- Improves file submission with direct upload forms

### Marmoset Theme Customizer
- Adds color customization panel (floating right sidebar)
- Choose from 8 premade themes (4 dark, 4 light)
- Customize individual colors: background, text, headers, accents, etc.
- Settings persist between sessions

## Usage
Both scripts will automatically activate on Marmoset pages. The theme customizer appears as a collapsible "Colours" panel on the right side of the screen.
Both scripts can run simultaneously without conflicts.
